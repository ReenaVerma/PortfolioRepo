project 1: sci-fi movie quiz

tech: Javascript, jQuery, Bulma

My first project at General Assembly, where I was required to build a game using JavaScript fundamentals I had learned over the last two weeks. I decided to pick a theme personal to me - sci-fi movies! Built using vanilla JavaScript and jQuery, the user is presented with a jumbled array of images and has to guess as many movies as possible within 60 seconds.

The game also incoporates sound clips, which the user and replay for hints in each around. And a score boards, which increments by 1 point, every time the user guesses correctly.
